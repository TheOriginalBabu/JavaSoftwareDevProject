Classes
    - Controller: Main class that controls all other classes to run the program
        Fields:
            - Config
            - FileReader
            - FileWriter
            - Generator
            - Teacher
            - TeacherDuty
            - Duty
            - Time
            - Location
            - Restriction
        Methods:
            - Controller
    - Window: Controls GUI
        Fields:
            - All GUI elements
            - arrays/Hashmaps of all objects
        Methods:
            - Window
            - main
            - windowInit
            - windowClose
            - jButtonInit(JButton)
            - jListInit(JList)
            - cardLayoutInit
            - setUpPageInit
    - Config: Stores all information about the program
        Fields:
            - ConfigFilePath
            - StorageFilePath
            - BackupFilePath
            - LastDateOpened
            - BackupDate
            - OpenNumber
        Methods:
            - Config
            - setConfigFilePath
            - getConfigFilePath
                - Returns a string
            - setStorageFilePath
            - getStorageFilePath
                - Returns a string
            - setBackupFilePath
            - getBackupFilePath
                - Returns a string
            - getLastDateOpened
            - setLastDateOpened
            - getBackupDate
                - Returns an Date object
            - setBackupDate
            - getOpenNumber
                - Returns an Date object
            - setOpenNumber
    - FileReader: Reads information from files
        Fields:
            - configPath
            - backUpPath
            - StoragePath
        Methods:
            - FileReader
                - returns information read
            - readConfig
                - returns information read
            - readBackup
                - returns information read
            - readStorage
                - returns information read
    - FileWriter: Writes information to files
        Fields:
            - configPath
            - backUpPath
            - StoragePath
        Methods:
            - FileWriter
            - writeConfig
            - writeBackup
            - writeStorage
    - Generator: Generates a schedule
        Fields:
            - Restrictions
        Methods:
            - Generator(restrictions)
            - generateWeek
            - generateDay
            - generatePeriod
    - Duty: Stores information about a duty
        Fields:
            - Time
            - Location
        Methods:
            - Duty(Time, Location)
            - getDuty
                - Returns a Time object
            - setDuty
            - setTime(Time)
            - getTime
                - Returns a Time object
            - setLocation(Location)
            - getLocation
                - Returns a Location object
    - TeacherDuty: Stores information about a teacher's duty
        Fields:
            - Teacher
            - Duty
        Methods:
            - TeacherDuty(teacher, duty)
            - getTeacher
                - Returns a Teacher object
            - setTeacher(Teacher)
            - setDuty(Duty)
            - getDuty
                - Returns a Duty object
    - Teacher: Stores information about a teacher
        Fields:
            - Name
            - Classes
            - PrepPeriods
            - Restrictions
            - MinutesTotal
            - MinutesUsed
            - MinutesRemaining
            - minutesScheduled
        Methods:
            - Teacher(string, Time, Time, minutes, restrictions)
            - getName
                - Returns a string
            - setName(string)
            - getClasses
                - Returns an array of Time objects
            - setClasses(Time)
            - getPrepPeriods
                - Returns an array of Time objects
            - setPrepPeriods(Time)
            - setMinutes(int)
            - getMinutes
                - Returns an int
            - setRestriction(Restriction)
            - getRestrictions
                - Returns an array of Restriction objects
    - Time: Stores information about a time
        Fields:
            - Name
            - startTime
            - endTime
        Methods:
            - Time(string)
            - getTime
                - Returns a string
            - setTime(string)
    - Location: Stores information about a location
        Fields:
            - Name
            - Description
            - TimesToWatch
        Methods:
            - Location(name, description)
            - getLocation
                - Returns a string
            - setLocation(string)
            - getDescription
                - Returns a string
            - setDescription(string)
            - setTimesToWatch(Time)
            - getTimesToWatch
                - Returns an array of Time objects
    - Restriction: Stores information about a restriction
        Fields:
                - Teacher
                - Time
                - Date
                - availableBool
        Methods:
            - Restriction(Teacher, Time, availableBool, Date)
            - getTeacher
                - Returns a Teacher object
            - setTeacher(Teacher)
            - getAvailableBool
                - Returns a boolean
            - setAvailableBool
            - getTime
                - Returns a Time object
